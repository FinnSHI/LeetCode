## 注意

- 注意 == 和 equals 的区别

  两个栈顶元素进行比较时，如果是对象，就要用equals，不能用==

  - ==
    - 基本数据类型：比较值是否相等
    - 引用类型：比较两个地址值
  - equals
    - Object类定义的equals是比较两个地址值是否相等
    - 包装类、String类、File、Date重写了equals方法
